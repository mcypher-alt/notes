# Заметки

Приложение для создания, хранения и управления заметками. Все данные сохраняются в `localStorage` браузера, поэтому они не теряются после закрытия страницы. Поддерживается поиск, тёмная тема и адаптивная вёрстка.

[Демо](https://mcypher-alt.github.io/notes/)

## Функционал

- Создание заметок с заголовком и текстом.
- Редактирование существующих заметок.
- Удаление заметок с подтверждением действия.
- Поиск по заголовкам заметок.
- Переключение светлой и тёмной темы оформления.
- Адаптивный интерфейс для компьютеров, планшетов и телефонов.
- Сохранение всех заметок и выбранной темы в `localStorage`.

## Технологии

- **HTML5** – семантическая разметка.
- **CSS3** – кастомные стили, адаптивная сетка (Grid и Flexbox).
- **JavaScript** – модульная архитектура, работа с DOM, события, `localStorage`.
- **Git** – система контроля версий.
